## Reporting a Vulnerability

We take the security of our setup scripts seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not open a public GitHub issue.**
2. Send an email to [INSERT EMAIL] with the details of the vulnerability.
3. Include a description of the issue, steps to reproduce, and the potential impact.

We will acknowledge receipt of your report within 48 hours and provide a timeline for a fix if necessary.
